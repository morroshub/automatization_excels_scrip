# Información del Proyecto

## Descripción
Este proyecto contiene una serie de archivos y scripts que te permiten generar informes a partir de archivos Excel configurables. Los informes se generan y se organizan por mes.

## Archivos y Directorios
Dentro de este proyecto encontrarás los siguientes elementos:

- **datasets**: Contiene los archivos de datos que puedes utilizar como entrada para los informes.

- **Jupyter notebook**: Un directorio que contiene Jupyter Notebooks relacionados con el proyecto.

- **Python script**: El script principal de Python que se utiliza para generar los informes a partir de los archivos Excel.

- **Requirements.txt**: Un archivo que lista las dependencias de Python necesarias para ejecutar el proyecto. Puedes instalar estas dependencias con `pip` utilizando el comando `pip install -r requirements.txt`.
